# Agent Instructions

This is a sandbox repository for experimenting with AI CLI coding agents. When working in this repo, follow the guidelines below.

## Repository Purpose

Use this repository to:
- Test and evaluate AI coding agents (Opencode, Claude Code, Aider, etc.)
- Run sandbox experiments without risk to production code
- Compare agent capabilities across tasks in `experiments/`

## Working in This Repo

- All experimental code lives in the `experiments/` directory.
- Each experiment has its own subdirectory with a `README.md` describing the task.
- Feel free to create, modify, and delete files inside `experiments/` freely.
- Do **not** modify `AGENTS.md`, `.opencode.json`, or `README.md` unless explicitly instructed.

## Code Style

- Use clear, readable code with descriptive variable names.
- Add comments when logic is non-obvious.
- Prefer small, focused functions.

## Commit Conventions

- Use conventional commit messages: `feat:`, `fix:`, `chore:`, `docs:`, `refactor:`, `test:`
- Keep commits small and focused.

## How to Complete an Experiment

1. Read the task description in the experiment's `README.md`.
2. Implement the solution in the experiment directory.
3. Verify your solution works as described.
4. Summarize what you did in a comment or short note.
